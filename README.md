# Django Logging System

It is a complete Logging system in Django that has benefits like :

- Log all request's headers that comes toward the app
- Log all queries in details
- Log all sessions that has been sign up or sign in

## My Experience

For the first time, when I tried to create this, I'd faced two issues : 

- <h2> How can you log all queries into log file? specially, If your project is in large scale? </h2>
- <h2> If you try to log all actions, you will have an overload on server, no doubt!!! </h2>
